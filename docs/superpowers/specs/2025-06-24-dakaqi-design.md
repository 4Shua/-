# 乱七八糟打卡器 — 设计规格

> 归档日期：2025-06-24 | 完整计划见 Cursor plans

## 产品定位

仿 HabitKit 的自用习惯打卡 App，纯本地、零成本，Android 优先。

## 核心功能（MVP）

1. **首页**：习惯卡片、类型筛选、分段圆环打卡、按月热力图
2. **新建习惯**：名称/描述/图标/颜色；高级选项（周期 n 次、打卡日、类型标签）
3. **详情浮层**：月历只读 + 月份切换 + 编辑入口
4. **设置**：关于/版本（极简）

## 关键交互

- 圆环按 n 分割，依次点亮，满格再点重置
- 热力图按**月份**分组（非 7 行周视图）
- 打卡日：每天 / 工作日 / 周末 / 法定节假日

## 明确不做

云同步、PRO、底部导航、年度热力图、提醒、Streak Goal

## 技术栈

Flutter + Drift + Riverpod
